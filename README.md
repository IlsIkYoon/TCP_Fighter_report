<h1>프로젝트 개요</h1>
<ul>Select 모델 학습 과정에서 만들었던 MMORPG를 가정한 전투 게임입니다.</ul>
<ul>섹터를 이용해 존 방식의 게임을 구현했으며 생성, 이동, 공격, 데미지, 사망을 메세지를 이용해 클라이언트들끼리 동기화했습니다.</ul>
<ul>학원 수업 과정에서 만들었으며, 클라이언트와 프로토콜은 제공받아 사용했습니다. 그 외는 모두 직접 구현했습니다.</ul>

<h2>트러블 슈팅 내용</h2>
<ul>유령 세션이 남는 버그를 발견해 덤프 메모리 확인 후 생성, 삭제 메세지를 컨테이너 이용해 카운팅 하는 방식으로 추적해서 해결했습니다.</ul>
